LAYOUT PATTERNS
================================================================================

1. HERO SECTIONS:
   - Full-width with gradient background
   - Centered content with max-width container
   - Responsive padding (px-4 sm:px-6 lg:px-8)
   - Minimum height constraints
   - Overlay effects with animated gradients

2. CONTENT SECTIONS:
   - Max-width containers (max-w-7xl, max-w-6xl, max-w-4xl)
   - Consistent padding (py-12 sm:py-20)
   - Grid layouts for multi-column content
   - Responsive breakpoints: sm (640px), md (768px), lg (1024px)

3. CARD LAYOUTS:
   - Rounded corners (rounded-2xl, rounded-3xl)
   - Border styling (border border-white/10)
   - Background overlays (bg-white/5, bg-[#01030B]/95)
   - Shadow effects (shadow-[0_25px_70px_rgba(0,0,0,0.6)])
   - Hover effects with scale and border color changes

4. GRID SYSTEMS:
   - 1 column on mobile
   - 2 columns on sm breakpoint
   - 3 columns on lg breakpoint
   - Gap spacing: gap-4, gap-6, gap-8

5. TYPOGRAPHY HIERARCHY:
   - H1: text-4xl md:text-5xl md:text-6xl (font-extrabold)
   - H2: text-3xl md:text-4xl (font-bold)
   - H3: text-2xl (font-bold)
   - Body: text-base, text-lg (text-slate-200, text-slate-300)
   - Small: text-sm, text-xs (text-slate-400)

================================================================================
ANIMATION SYSTEM
================================================================================

1. PAGE ANIMATIONS:
   - Fade-in on load
   - Stagger children animations
   - Scroll-triggered animations (whileInView)

2. COMPONENT ANIMATIONS:
   - Hover scale effects
   - Border animations
   - Gradient shifts
   - Floating animations

3. CUSTOM KEYFRAMES:
   - fadeIn: Opacity 0 to 1
   - slideUp: TranslateY 20px to 0
   - slideDown: TranslateY -20px to 0
   - scaleIn: Scale 0.95 to 1
   - glow: Box shadow pulse
   - pulseRing: Scale and opacity pulse
   - pulseGlow: Drop shadow pulse

4. TRANSITION TIMINGS:
   - Fast: 0.2s - 0.3s (hover effects)
   - Medium: 0.4s - 0.6s (entrance animations)
   - Slow: 0.8s - 1.2s (complex animations)
